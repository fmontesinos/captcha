Implementa RGPD en tu Sitio Web

Con captcha.gratis, añadir un sistema de gestión de cookies compatible con el RGPD es fácil y gratuito. Sigue estos pasos para que tu sitio cumpla con la normativa de protección de datos y ofrezca una mejor experiencia a tus usuarios. Ver o descargar este ejemplo en GitHub.

¿Qué es el RGPD y por qué lo necesitas?

El RGPD (Reglamento General de Protección de Datos) es una ley que protege la privacidad de los usuarios en internet. Si tu sitio web usa cookies (por ejemplo, para analizar visitas o mostrar anuncios), necesitas pedir permiso a tus visitantes. Nuestro script te ayuda a hacerlo de forma sencilla, mostrando un modal donde los usuarios pueden elegir qué cookies aceptar.

Cómo añadir el sistema RGPD a tu web

Paso 1: Copia el código

Añade este código al final del <body> de tu página HTML. Este script muestra un modal de cookies a tus visitantes. Usa token=test para pruebas.

<script src="https://captcha.gratis/rgpd.js.php?lang=es&force=true&policyUrl=/politica-cookies&token=test"></script>

Paso 2: Obtén un token para tu sitio

El parámetro token identifica tu sitio web. Para pruebas, usa token=test. Cuando estés listo para usar el script en tu sitio real, regístrate en captcha.gratis para obtener un token único. Luego, reemplaza test con tu token. Ejemplo:

<script src="https://captcha.gratis/rgpd.js.php?lang=es&force=true&policyUrl=/politica-cookies&token=TU_TOKEN_AQUI"></script>

Paso 3: Personaliza el idioma (opcional)

El parámetro lang controla el idioma del modal. Usa es para español, en para inglés, o fr para francés. Por ejemplo, para inglés:

<script src="https://captcha.gratis/rgpd.js.php?lang=en&force=true&policyUrl=/politica-cookies&token=test"></script>

Paso 4: Añade acciones basadas en el consentimiento (opcional)

Si usas herramientas como Google Analytics o anuncios, puedes cargarlas solo si el usuario da su consentimiento. Añade este código antes del script RGPD:

<script>
  window.onConsentChange = function(preferences) {
    console.log('Consentimiento actualizado:', preferences);
    if (preferences.cookies.analytics) {
      console.log('Cargando Google Analytics...');
      // Aquí iría el código para cargar Google Analytics
    }
    if (preferences.cookies.advertising) {
      console.log('Cargando anuncios...');
      // Aquí iría el código para cargar anuncios
    }
  };
</script>

Paso 5: Prueba el modal

Haz clic en el botón de abajo para ver cómo aparece el modal de cookies en tu sitio. Puedes probarlo en diferentes idiomas cambiando el parámetro lang.

¿Dónde pegar el código?

Si usas un editor web (como WordPress, Wix, o un editor HTML), busca la sección donde puedes añadir código personalizado. Pega el script justo antes de la etiqueta </body>. Por ejemplo:

<body>
  <h1>Mi Sitio Web</h1>
  <p>Bienvenido a mi sitio.</p>

  <script src="https://captcha.gratis/rgpd.js.php?lang=es&force=true&policyUrl=/politica-cookies&token=test"></script>
</body>
